# Buzz Reports

This repo contains only the reports module frontend extracted from local HRMS.

## Files

- `index.html` (mount + payload + script includes)
- `static/js/reports_react.js`
- `static/css/reports_react.css`
- `vercel.json`

## Deploy on Vercel

1. Import this repository in Vercel.
2. Framework preset: `Other`.
3. Build Command: leave empty.
4. Output Directory: `.` (root).
5. Deploy.

The UI and behavior match local reports at `http://127.0.0.1:8030/reports/?v=48`.
